NAME
	meminf - prints system memory information depending on switches.

SYNOPSIS
	meminf [OPTION]...
	meminf [OPTION 1]... [OPTION 2]... [OPTION N]...	

DESCRIPTION
	Lists information about the system's memory according to the linux /proc/meminfo
	file. Information for RAM or other volatile memory will fluctuate with restarting 
	the shell but not while within the shell.

	List of availible switches are:
	
	-a (List all memory information availible)
	
	-s (Lists the available free swap space in bytes)

	-S (Lists the available total shared memory in bytes)

	-u (Lists the available free memory in bytes)

EXIT CODES
	0	is OK,

	1	if an unknown switch given,

	2	if invalid syntax in parameters.
